🏥 AI-Powered Voice-Based Appointment Booking System
📌 Overview

This project is an AI-powered system designed to automate patient interaction, doctor recommendation, and appointment booking using intelligent workflows.

It simplifies the traditional healthcare booking process by allowing users to describe their symptoms and automatically:

Identifies the appropriate doctor type
Checks doctor availability
Books appointments efficiently
🚀 Features
🤖 AI-Based Symptom Analysis
Users input their symptoms
System suggests the relevant doctor specialization
📅 Smart Appointment Booking
Fetches available time slots
Allows booking based on doctor availability
📊 Dashboard
Displays:
Total patients
Successful bookings
⚙️ Backend (FastAPI)
REST APIs for:
Booking
Metrics
Patient handling
🧠 Scalable Architecture
Designed for future integration with:
AI voice agents
Insurance verification
Multi-agent workflows
